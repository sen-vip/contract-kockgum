# 계약콕검 v0.1

공사·용역·물품 계약서류를 `서울특별시교육청 계약업무 처리지침(2023.12.27. 개정) [붙임4] 계약 구비서류` 순서대로 확인하는 정적 웹앱입니다.

## 주요 기능

- 공사 → 용역 → 물품 순서의 계약유형 탭
- 붙임4 순서 기반 단계별 체크리스트
- 상태값: 미확인 / 확인완료 / 첨부확인 / 생략가능 / 해당없음 / 보류
- 파일명 기반 자동 매칭
- 수의계약 통합서약서 자동 반영: 청렴서약서, 수의계약 체결제한 여부 확인서, 수의계약 각서, 조세포탈 서약서를 첨부확인 처리
- 스캔 PDF·이미지 OCR 제목 확인 보조
- 미분류 파일 수동 연결
- 진행률/요약
- 인쇄용 확인결과
- JSON 저장/불러오기
- localStorage 자동저장

## 개인정보·파일 처리

- 파일은 서버로 전송하지 않습니다.
- 브라우저 내부에서 파일명 및 OCR 제목 확인만 수행합니다.
- 새로고침 후에는 보안상 원본 파일 접근 권한이 사라지므로 OCR 재확인이 필요하면 파일을 다시 선택해야 합니다.
