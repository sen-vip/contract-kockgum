# 계약콕검 v0.1.6

공사·용역·물품 계약서류를 `서울특별시교육청 계약업무 처리지침(2023.12.27. 개정) [붙임4] 계약 구비서류` 순서대로 확인하는 정적 웹앱입니다.

## 주요 기능

- 공사 → 용역 → 물품 순서의 계약유형 탭
- 붙임4 순서 기반 단계별 체크리스트
- 상태값: 미확인 / 확인완료 / 첨부확인 / 생략가능 / 해당없음 / 보류
- 파일명 기반 자동 매칭 및 즉시 첨부확인 반영
- 수의계약 통합서약서 자동 반영: 청렴서약서, 수의계약 체결제한 여부 확인서, 수의계약 각서, 조세포탈 서약서를 첨부확인 처리
- 스캔 PDF·이미지 OCR 제목 확인 보조
- 보완서류처럼 여러 장이 합쳐진 PDF의 페이지별 제목 OCR 확인
- 자동분류된 파일은 수동 연결창 숨김, 미분류 파일만 수동 연결
- 진행률/요약
- 인쇄용 확인결과
- JSON 저장/불러오기
- localStorage 자동저장
- 계약금액 세 자리 콤마 자동 입력
- 자동반영/수동확인/생략가능/해당없음/보류 상태별 색상 구분
- 파일 첨부 후 자동분류 결과 영역으로 자동 스크롤
- 파일 첨부 결과 목록 컴팩트 표시 및 내부 스크롤
- 접힌 카드에서도 확인/생략/해당없음/보류 빠른 상태 변경
- 상단 바로가기 클릭 시 단계 제목 위치로 정확히 이동
- 푸터 제작자·GitHub 링크 표시
- 맨위로 바로가기
- 서류만 초기화 / 계약정보 초기화 / 전체 초기화 분리
- PDF 페이지별 OCR 결과에 페이지번호와 연결 서류 표시

## 개인정보·파일 처리

- 파일은 서버로 전송하지 않습니다.
- 브라우저 내부에서 파일명 및 OCR 제목 확인만 수행합니다.
- 새로고침 후에는 보안상 원본 파일 접근 권한이 사라지므로 OCR 재확인이 필요하면 파일을 다시 선택해야 합니다.

## 배포

정적 파일만 사용하므로 GitHub Pages, Vercel 등에 그대로 배포할 수 있습니다.

```text

README.md
```

## 주의

본 도구는 서울특별시교육청 계약업무 처리지침(2023.12.27. 개정) [붙임4] 계약 구비서류를 바탕으로 한 서류 확인 보조도구입니다. 계약금액, 계약방법, 사업 성격, 최신 지침 개정 여부에 따라 실제 징구 서류가 달라질 수 있으므로 최종 확인은 담당자가 직접 수행해야 합니다.

## v0.1.6 업데이트

- 상단 바로가기 영역에 `맨위로` 버튼을 추가했습니다.
- 초기화를 `서류만 초기화`, `계약정보 초기화`, `전체 초기화`로 분리했습니다.
- 보완서류처럼 여러 장이 합쳐진 PDF를 페이지별로 OCR 확인합니다.
- OCR 결과에 `파일명 · 페이지번호 · 인식 제목 · 연결 서류`를 표시합니다.
- `일반 산업재해 예방 체크리스트`, `예정 공정표` 같은 실제 제출 제목을 붙임4 서류명에 연결하는 유사어 매칭을 보강했습니다.
