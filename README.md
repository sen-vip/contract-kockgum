# 계약콕검 v0.1.9

공사·용역·물품 계약서류를 `서울특별시교육청 계약업무 처리지침(2023.12.27. 개정) [붙임4] 계약 구비서류` 순서대로 확인하는 정적 웹앱입니다.

## 주요 기능

- 공사 → 용역 → 물품 순서의 계약유형 탭
- 붙임4 순서 기반 단계별 체크리스트
- 상태값: 미확인 / 확인완료 / 첨부확인 / 생략가능 / 해당없음 / 보류
- 파일명 기반 자동 매칭 및 즉시 첨부확인 반영
- 수의계약 통합서약서 자동 반영: 청렴서약서, 수의계약 체결제한 여부 확인서, 수의계약 각서, 조세포탈 서약서를 첨부확인 처리
- 스캔 PDF·이미지 OCR 제목 확인 보조
- 보완서류처럼 여러 장이 합쳐진 PDF의 페이지별 제목 OCR 확인
- 자동분류된 파일은 수동 연결창 숨김, 미분류 파일만 수동 연결
- 진행률/요약
- 인쇄용 확인결과
- JSON 저장/불러오기
- localStorage 자동저장
- 계약금액 세 자리 콤마 자동 입력
- 자동반영/수동확인/생략가능/해당없음/보류 상태별 색상 구분
- 파일 첨부 후 자동분류 결과 영역으로 자동 스크롤
- 파일 첨부 결과 목록 컴팩트 표시 및 내부 스크롤
- 접힌 카드에서도 확인/생략/해당없음/보류 빠른 상태 변경
- 상단 바로가기 클릭 시 단계 제목 위치로 정확히 이동
- 푸터 제작자·GitHub 링크 표시
- 맨위로 바로가기
- 서류만 초기화 / 계약정보 초기화 / 전체 초기화 분리
- PDF 페이지별 OCR 결과에 페이지번호와 연결 서류 표시

## 개인정보·파일 처리

- 파일은 서버로 전송하지 않습니다.
- 브라우저 내부에서 파일명 및 OCR 제목 확인만 수행합니다.
- 새로고침 후에는 보안상 원본 파일 접근 권한이 사라지므로 OCR 재확인이 필요하면 파일을 다시 선택해야 합니다.

## 배포

정적 파일만 사용하므로 GitHub Pages, Vercel 등에 그대로 배포할 수 있습니다.

```text
index.html
style.css
app.js
README.md
```

## 주의

본 도구는 서울특별시교육청 계약업무 처리지침(2023.12.27. 개정) [붙임4] 계약 구비서류를 바탕으로 한 서류 확인 보조도구입니다. 계약금액, 계약방법, 사업 성격, 최신 지침 개정 여부에 따라 실제 징구 서류가 달라질 수 있으므로 최종 확인은 담당자가 직접 수행해야 합니다.


## v0.1.9 업데이트

- 스캔서류 제목 판독 보정을 강화했습니다.
- PDF OCR 영역을 상단/제목영역/중앙영역/본문단서로 나누어 읽습니다.
- OCR 전 이미지 확대, 대비 강화, 배경 정리를 보강했습니다.
- `전문건설업등록증`이 흐리게 읽히는 경우 `별지 제3호 서식 + 건설업 등록 단서`로 `공사 면허 등록증, 등록수첩` 확인필요까지 잡도록 개선했습니다.
- `사업자등록증`은 제목 직접 인식 또는 `법인사업자`, `사업자등록번호`, `개업연월일`, `사업장 소재지` 등 복수 단서가 있을 때만 확인필요로 제안합니다.
- 애매한 페이지별 OCR 결과에 `반영` 버튼을 추가해, 확인필요 항목을 바로 체크리스트에 연결할 수 있게 했습니다.
- `인감증명서`가 사업자등록증으로 잘못 붙지 않도록 인감증명서 우선 판정과 오분류 방지 규칙을 유지·강화했습니다.

## v0.1.8 업데이트

- OCR 확인 중 상태를 `OCR 확인 중.` → `OCR 확인 중..` → `OCR 확인 중...`처럼 움직이게 했습니다.
- 파일별 OCR 진행률을 `3/12쪽 확인 중`과 진행바로 표시합니다.
- PDF 페이지별 OCR을 상단 넓은 영역과 제목 후보 영역으로 나누어 확인하도록 보강했습니다.
- 흐린 스캔 제목을 위해 OCR용 대비 보정을 추가했습니다.
- `전문건설업등록증`, `사업자등록증`, `인감증명서`, `사용인감계` 매칭을 보강했습니다.
- `등록번호`, `증명서`, `법인`, `제출용` 같은 단독 키워드로는 자동분류하지 않도록 했습니다.
- 인감증명서가 사업자등록증 사본으로 잘못 붙는 문제를 줄이도록 오분류 방지 규칙을 추가했습니다.
- 애매한 OCR 결과는 자동반영하지 않고 `확인필요`로 남기도록 했습니다.
