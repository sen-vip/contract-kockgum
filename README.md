# 계약콕검 v0.1.11

공사·용역·물품 계약서류를 `서울특별시교육청 계약업무 처리지침(2023.12.27. 개정) [붙임4] 계약 구비서류` 순서대로 확인하는 정적 웹앱입니다.

## v0.1.11 핵심 변경

- OCR 페이지별 결과에서 `계약 체결 ·` 같은 단계명은 빼고 **반영될 서류명만 바로 보이도록** 정리했습니다.
- 긴 OCR 제목과 긴 서류명은 hover 시 전체가 펼쳐져 보이도록 개선했습니다.
- 페이지 결과 행에 tooltip을 추가해 잘린 서류명을 마우스 올림으로 확인할 수 있습니다.
- v0.1.10의 빠른 OCR 기본/정밀 OCR 선택 구조는 유지했습니다.

## 주요 기능

- 공사 → 용역 → 물품 순서의 계약유형 탭
- 붙임4 순서 기반 단계별 체크리스트
- 상태값: 미확인 / 확인완료 / 첨부확인 / 생략가능 / 해당없음 / 보류
- 파일명 기반 자동 매칭 및 즉시 첨부확인 반영
- 수의계약 통합서약서 자동 반영: 청렴서약서, 수의계약 체결제한 여부 확인서, 수의계약 각서, 조세포탈 서약서를 첨부확인 처리
- 보완서류처럼 여러 장이 합쳐진 PDF의 페이지별 제목 확인
- 빠른 OCR 기본 실행 + 실패 페이지 정밀 OCR 선택 실행
- OCR 결과에서 서류명 우선 표시 및 hover 전체보기
- 자동분류된 파일은 수동 연결창 숨김, 미분류 파일만 수동 연결
- 진행률/요약
- 인쇄용 확인결과
- JSON 저장/불러오기
- localStorage 자동저장
- 계약금액 세 자리 콤마 자동 입력
- 자동반영/수동확인/생략가능/해당없음/보류 상태별 색상 구분
- 파일 첨부 후 자동분류 결과 영역으로 자동 스크롤
- 접힌 카드에서도 확인/생략/해당없음/보류 빠른 상태 변경
- 상단 바로가기 클릭 시 단계 제목 위치로 정확히 이동
- 맨위로 바로가기
- 서류만 초기화 / 계약정보 초기화 / 전체 초기화 분리
- 푸터 제작자·GitHub 링크 표시

## 개인정보·파일 처리

- 파일은 서버로 전송하지 않습니다.
- 브라우저 내부에서 파일명 및 OCR 제목 확인만 수행합니다.
- 새로고침 후에는 보안상 원본 파일 접근 권한이 사라지므로 OCR 재확인이 필요하면 파일을 다시 선택해야 합니다.
